# Clinical-Trial-Data-Analysis
This project demonstrates an end-to-end data analytics workflow using a simulated clinical trial dataset. The objective is to extract meaningful insights from patient, hospital, and treatment data while applying industry-standard data analysis techniques.

**Problem Statement**

Clinical trials generate large amounts of patient, treatment, and hospital data. Without proper analysis, it is difficult to determine which treatment groups are most effective, identify adverse events, monitor patient outcomes, control treatment costs, and evaluate trial performance across different countries and study sites.

The objective of this project is to integrate patient, treatment, and hospital datasets to identify actionable insights that improve treatment effectiveness, patient safety, trial efficiency, and healthcare decision-making.

**Objectives**

1. Measure the effectiveness of each treatment group.

2. Compare patient outcomes across diseases.

3. Identify treatments associated with higher adverse events.

4. Analyze treatment costs across trial groups.

5. Monitor trial completion and withdrawal rates.

6. Evaluate study site performance.

7. Understand patient demographics participating in trials.

8. Identify high-risk patient groups requiring closer monitoring.

9. Measure follow-up compliance across hospitals.

10. Support data-driven decisions for future clinical trials.

# **Key Performance Indicators (KPIs)**

**Patient KPIs**

- Total Patients Enrolled

- Average Patient Age

- Gender Distribution

- Disease-wise Patient Count

- Average BMI

- High-Risk Patient Percentage (e.g., BMI > 30)
